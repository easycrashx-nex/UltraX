# UltraX Browser 1.2.2

Internal version: `1.2.2`.

Release tag: `v1.2.2`.

## HTML fullscreen fix

- Fixed the fullscreen button on YouTube, Twitch, and other HTML5 media pages.
- Chromium HTML fullscreen events are now handled by the frameless UltraX window.
- The active page expands over the browser chrome while fullscreen is active.
- Exiting fullscreen restores the normal tab bar, toolbar, bookmarks bar, and view bounds.
