# UltraX Browser 1.1.9-Fix

This targeted fix build preserves the UltraX 1.1.9 feature set and corrects two browser-chrome layout regressions.

- Quick Settings now reserves its complete native overlay region on remote pages.
- Quick Settings has a viewport-safe scroll area, sticky header, and accessible footer action.
- New Tab transitions detach stale remote `WebContentsView` instances cleanly.
- Address suggestions stay anchored below the address bar, scroll internally, and do not push New Tab content.
- Resize, maximize, side-panel, download-shelf, and lifecycle bounds are calculated centrally.

Visible product label: `UltraX Browser 1.1.9-Fix`.
Internal package version: `1.1.9-fix.1`.
Preferred release tag: `v1.1.9-Fix`.
